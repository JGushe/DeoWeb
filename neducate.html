Cómo usar un computador

En este pequeño curso de 5 clases aprenderas como usar un computador, navegar en el entorno virtual de un sistema operativo y diversos trucos más.

En este curso veremos:

  • Entorno de Windows, cosas clave, aplicaciones por defecto.
  • Como encender, apagar, suspender, perifericos, tipos de conexiones, entre otros.
  • Como crear carpetas, tipos de archivos, entre otros.
  • Como usar el navegador, tips, configuración
  • Como instalar aplicaciones, desinstalar aplicaciones, donde encontrarlas en internet.

Precio por Clase: $5

Instalación de Software y Sistemas Operativos

Te enseñaremos a instalar software, juegos, formatear discos, crear particiones, sistemas operativos (Windows o basados en Linux), multi-boot .

En este curso de 4 clases veremos:

  • Instalación de Sowtfare: Aplicaciones en general, juegos, donde conseguirlos, sowftware para empezar a programar, entre otros.
  • Manipulación de Almacenamiento: Activar discos, crear particiones, eliminar particiones, distribuir espacios y tipos de formatos.
  • Sistemas Operativos: Desinstalar sistemas operativos de nuestro computador, instalar Windows, instalar basados en Linux.
  • GRUB (Dual-Boot): Instalar dos o más sistemas operativos en un mismo disco, crear herramientas para instalar sistemas operativos (USB Multi-boot), configurar el GRUB de GNU/LINUX para acceder al sistema operativo que querramos.

Precio por Clase: $5

Mantenimiento habitual de un computador

Te enseñaremos como quitar suciedad de un computador, cambiar sus pastas termincas, agregar componentes, y demás

En este curso de 3 clases te enseñaremos:

  • Limpieza: Limpiar suciedad de un computador, que herramientas usar, que componentes son seguros retirar.
  • Mantenimiento: Cambiar pasta terminca del procesador, desensamblar componentes de la mainboard, limpiar contactos de manera segura
  • Agregar o retirar componentes: Te enseñaremos a retirar componentes computador, cambiar componentes y comprobar si nuestra computadora los soporta.

Elección de componentes

En este curso de 3 clases veremos
